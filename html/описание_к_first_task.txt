Первое задание будет работать в следующем случае:

1. В корневой директории сайта auslogics.com создать файл ".htaccess" (или дополнить файл) с содержанием, находящимся в файле ".htaccess", поставляемом к этому заданию.
2. На хостинг сайта auslogics.com поместить файл "php/first_task.php" по следующему пути "корневая_директория_сайта/Auslogics/test_tasks/php/first_task.php".
3. В корневой директории сайта auslogics.com расположить файл "file.exe".
4. На любом другом хостинге с доменным именем необходимо запустить файл "html/first_task.html".
